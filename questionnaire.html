1. Différence entre MongoDB et PostgreSQL MongoDB est une base NoSQL orientée documents (JSON/BSON, schéma flexible). PostgreSQL est une base relationnelle SQL avec schéma strict et relations (tables, clés). 2. Exemple simple de structure de données (utilisateur) { "id": 1, "nom": "Dupont", "prenom": "Jean", "email": "jean@exemple.com" } 3. Rôle d’une clé primaire Identifier de manière unique chaque enregistrement dans une table/collection et permettre les relations entre données. 4. Organisation des données utilisateurs / missions Utilisateurs : table/collection users (id, nom, email, …) Missions : table/collection missions (id, titre, description, user_id) Chaque mission référence l’utilisateur via user_id (relation 1 utilisateur → plusieurs missions).
